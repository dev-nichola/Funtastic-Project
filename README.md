# Funtastic Project
## Install Project
```
composer create-project nichola/funtastic
```
## Install Composer
```
composer install
```

## Install NPM
```
npm install
```
## Run Tailwindcss
```
npm run build:css
```
## Setting env
you can copy .env.example to .env
```
APP_URL = localhost
APP_PORT = 8100

DB_HOST =localhost
DB_PORT =3306
DB_DATABASE =dev_php_test
DB_USERNAME =root
DB_PASSWORD =123

```

## Run Project
```
php funtastic.php
```
